# Introduction to Git and GitHub - Final Project

This repository contains the final project for the "Introduction to Git and GitHub" course. The project demonstrates a practical understanding of version control workflows, using both the GitHub User Interface (UI) and the Git Command Line Interface (CLI).

## Project Overview

The objective of this project is to practice and showcase essential DevOps skills including:
* Forking and cloning repositories.
* Managing branches and collaboration files (License, Code of Conduct, Contributing guidelines).
* Implementing script-based calculators (`simple-interest.sh`).
* Handling branch merges, resolving conflicts, and working with tracking branches via Git CLI.

## Repository Structure

* `README.md` - Project documentation and overview.
* `LICENSE` - Apache 2.0 open-source license details.
* `CODE_OF_CONDUCT.md` - Standards and pledge for community engagement.
* `CONTRIBUTING.md` - Guidelines for open-source contributions and bug reporting.
* `simple-interest.sh` - A Bash script that calculates simple interest based on principal, annual rate of interest, and time period.
